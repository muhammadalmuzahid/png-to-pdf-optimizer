# PNG to Optimized PDF Converter

A Python-based utility to merge multiple PNG images into a single, high-quality, optimized PDF. 

## Features
- **High Resolution:** Defaults to 300 DPI for professional print quality.
- **Optimized Size:** Uses smart JPEG compression to reduce file size significantly without visible quality loss.
- **Sharp Edges:** Disables chroma subsampling to keep text and lines crisp even after compression.

## Requirements
- Python 3.x
- Pillow library
